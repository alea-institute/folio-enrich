    Create `scripts/contrast-audit.mjs` as an ES module Node.js script (shebang `#!/usr/bin/env node`) implementing WCAG contrast math and CSS parsing.

    **Required exports (for testability):**
    ```
    export function hexToRgb(hex)                            // "#rrggbb" -> {r,g,b}
    export function relativeLuminance(rgb)                   // WCAG formula: 0.2126*R + 0.7152*G + 0.0722*B after gamma correction
    export function contrastRatio(fgHex, bgHex)              // (L1+0.05)/(L2+0.05)
    export function mixColors(fgHex, bgHex, fgFraction)      // alpha-composite fg onto bg
    export function parseCssVariables(cssBlock)              // parses `--name: value;` lines into a Map
    export function resolveVariable(name, themeMap, paletteMap)  // follows var(--x) chains to hex
    export async function runAudit()                         // reads index.html, builds report, writes file
    ```

    **WCAG formulas (from https://www.w3.org/TR/WCAG21/#contrast-minimum):**
    - sRGB channel to linear: if c <= 0.03928: c/12.92 else ((c+0.055)/1.055)^2.4
    - Relative luminance: 0.2126*R_linear + 0.7152*G_linear + 0.0722*B_linear
    - Contrast ratio: (lighter + 0.05) / (darker + 0.05)

    **Parsing approach:**
    1. Read `frontend/index.html` via fs.readFileSync.
    2. Extract `:root { ... }` block (first occurrence) — contains palette primitives.
    3. Extract each `[data-theme="X"] { ... }` block via regex: `/\[data-theme="(dark|light|mixed)"\]\s*\{([^}]+)\}/g`
    4. Extract the `[data-theme="mixed"] .panel-right, ... { ... }` override block and treat as "mixed-light" context.
    5. For each block, parse `--name: value;` pairs. Values may be literal hex (`#rrggbb`), `var(--other)`, or `rgba(...)` (skip rgba for text/bg tokens).
    6. Build paletteMap from :root (all literal hex values).
    7. For each theme, build themeMap. When resolving a token, if value is `var(--x)`, look up `--x` in themeMap first, then paletteMap. Recurse up to depth 5.

    **Audit logic — runAudit():**
    For each theme ∈ {dark, light, mixed, mixed-light}:
      For each background token ∈ [--bg, --surface, --surface2, --surface3, --modal-bg, --tooltip-bg]:
        resolvedBg = resolveVariable(bg, themeMap, paletteMap)
        For each text token ∈ [--text, --text-dim, --accent, --accent-dim]:
          resolvedFg = resolveVariable(text, themeMap, paletteMap)
          ratio = contrastRatio(resolvedFg, resolvedBg)
          threshold = 4.5 (body text default)
          status = ratio >= 4.5 ? "PASS" : (ratio >= 3.0 ? "LARGE-ONLY" : "FAIL")
          record {theme, fg, bg, fgHex, bgHex, ratio, status}

      For each branch token (26 branches):
        resolvedBranch = resolveVariable(branch, themeMap, paletteMap)
        For each base bg ∈ [--bg, --surface]:
          resolvedBase = resolveVariable(baseBg, themeMap, paletteMap)
          effectiveBg = mixColors(resolvedBranch, resolvedBase, 0.12)   // 12% branch tint
          resolvedText = resolveVariable("--text", themeMap, paletteMap)
          ratio = contrastRatio(resolvedText, effectiveBg)
          threshold = 4.5
          record {theme, branch, baseBg, ratio, status}

    ```markdown
    **Generated:** {ISO date}

    ```
