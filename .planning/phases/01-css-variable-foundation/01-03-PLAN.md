**Step 1: Audit all remaining hardcoded colors in the ENTIRE file.**

Run this grep to find every remaining hardcoded hex color:
```bash
grep -nP '#[0-9a-fA-F]{3,8}' frontend/index.html | grep -v ':root'
```

Categorize each hit:
- **Inside `<style>` but outside `:root`**: Should have been caught by Plan 02. Convert to var().
- **In static HTML `style=""` attributes**: Convert to var() — inline styles CAN use CSS custom properties.
- **In JavaScript template literals**: Note but do NOT convert (Phase 2 scope). These are the BRANCH_COLORS object and `style="color:..."` in JS template literals.
- **In HTML comments or CSS comments**: Leave as-is.
- **In the `:root` palette block**: Expected — these are the primitives.

**Step 2: Convert any remaining static HTML inline style colors.**

Example conversions:
```html
<!-- Before -->
<div style="font-size:12px;color:#6b6b80;margin-bottom:10px">

<!-- After -->
<div style="font-size:12px;color:var(--text-dim);margin-bottom:10px">
```

Common patterns to find and convert:
- `style="...color:#1a1a2e..."` → `style="...color:var(--text)..."`
- `style="...color:#6b6b80..."` → `style="...color:var(--text-dim)..."`
- `style="...color:#8b8fa3..."` → `style="...color:var(--text-dim)..."`
- `style="...background:#ffffff..."` → `style="...background:var(--bg)..."`
- `style="...border-color:#d0d2da..."` → `style="...border-color:var(--border)..."`

**Step 3: Document the final color audit.**

After all conversions, run the final audit:
```bash
grep -nP '#[0-9a-fA-F]{3,8}' frontend/index.html | grep -v ':root' | grep -v '/\*'
```

The remaining hits should ALL be in JavaScript code (lines 2464+). Count and document them — these are Phase 2 scope.
